## 写博客

详见 [`data/posts/README.md`](data/posts/README.md)。简述：

1. 新建 `data/posts/你的slug.md`
2. 在 `data/posts.json` 加一条（title / date / summary / slug）
3. `git add` → `commit` → `push`
